# Tic-Tac-Toe-With-Testing
It is a command line tic tac toe game with Rspec testing.

### Built With
+ Ruby
+ RSpec

### Requirement
 + Ruby 2.5.0+
 + Rspec 3.8.0

### Run
 + bundle install
 + ruby lib/main.rb
